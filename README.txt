


                                P A I N T E R




  What is this?
  -------------

  Painter is a servlet for rendering map marker images to be used in Exhibit
  or elsewhere.


  Running Painter
  ----------------

  Painter can be run at the command line using Maven:
  
    mvn jetty:run
  
  Then you can access it through your browser at
  
   http://127.0.0.1:5555/painter/painter
